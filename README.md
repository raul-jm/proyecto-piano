# proyecto-piano
Proyecto de situación de aprendizaje para enseñanzas elementales de piano titulado “Escucharse para mejorar”. Fomenta la autoevaluación mediante grabaciones, desarrollando la escucha crítica y la autonomía. Incluye tareas secuenciadas, criterios de evaluación, rúbrica y guía didáctica creadas con eXeLearning.

Créditos del proyecto

Autoría: Raúl Jiménez Medina
Centro: C.P.M "Maestro Chicano", Lucena
Descripción:
Este recurso forma parte de una situación de aprendizaje titulada “Escucharse para mejorar”, orientada al alumnado de enseñanzas elementales de piano. El objetivo es fomentar la autoevaluación y la mejora interpretativa mediante el uso de grabaciones.
